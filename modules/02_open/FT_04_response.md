## FT-4 — Functional Response Principle
**Status:** A_DISC (Open)
**Expression:** `Response = R(Φ, ∇Φ, ∂tΦ, ρ, κ, m)`

---

### Statement

Every instrumental or material response depends on the complete
local field state — not on a single isolated parameter.

Nothing is isolable from the field that surrounds it.

---

### Parameters

**Φ** — local field state
**∇Φ** — spatial gradient
**∂tΦ** — temporal variation
**ρ** — local density
**κ** — local coherence
**m** — field memory

---

### Role in the Framework

FT-4 is a **conceptual compass**, not a calculatory equation.

It says: any response depends on the full local context.
It does not yet say how R factorizes —
which parameters dominate in which regime,
and whether R is separable or irreducibly coupled.

This is the open zone.

---

### Why A_DISC and Not Abandoned

Removing FT-4 would create a structural gap:
without it, FTs 1–3 and 14 describe field states and drivers,
but say nothing about how a system *responds* to those states.

FT-4 holds the place of a necessary formula
that is not yet fully formalized.

---

### Open Zone

**What remains to be determined:**
- Functional form of R: additive? multiplicative? hierarchical?
- Which parameters dominate in which regime?
  (Dense/slow: does m dominate? Diffuse/fast: does ∇Φ dominate?)
- Is R separable: R(Φ)·R(∇Φ)·R(κ)... or irreducibly coupled?

**Experimental path:**
A phenomenon where the same stimulus produces radically
different responses depending on the prior state of the system
— and where the responsible parameter can be identified.

---

### Observable Candidate

**PH-005 (mayonnaise):**
Same gesture, same ingredients, same temperature —
emulsion forms or fails depending on the prior coherence state κ
of the mixture.
The cook detects the optimal κ state empirically without naming it.

This is directly FT-4: the response (emulsification)
depends on a parameter (κ) that is invisible to the standard framework,
which predicts identical results from identical inputs.

**Discriminating test:** same ingredients, same protocol,
one starting mixture with coherent κ (pre-tempered oil, ingredients at same temperature),
one with incoherent κ (cold oil, warm egg) → systematic difference in outcome.

---

### Falsification Criterion

FT-4 fails if:
a response is demonstrated to be fully determined by a single parameter
in all documented cases — i.e., the full field context is always reducible
to one dominant variable.
