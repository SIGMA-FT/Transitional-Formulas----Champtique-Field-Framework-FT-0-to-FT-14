## FT-8 — Field Memory: Coherent Information Persists
**Status:** ACQUIRED
**Expression:** `∂tm = (1/τm(κ)) · (H(Φ,ρ,κ) − m)`

---

### Statement

The field carries memory of its past states.
The duration of that memory depends on the local coherence κ.

---

### Parameters

**m** — field memory state
**τm(κ)** — memory time constant — depends on κ
**H(Φ,ρ,κ)** — memory target function

---

### Critical Correction: τm is κ-Dependent

The earlier version with constant τm introduced spurious uniform dissipation
— incompatible with the Pch postulate of non-dissipation.

**High κ (coherent information):** τm → ∞ — memory is permanent
**Low κ (incoherent information):** τm finite — progressive dissolution

This is structurally consistent with the architecture of Φ global:
coherent information inscribes durably.
Incoherent information dissolves for lack of anchor
— unless it finds a vulnerable condensed substrate.

---

### Prototypical Form (A_FORMALISER)

```
τm(κ) = τ₀ · exp(λ · κ)
```

Constants τ₀ and λ to be extracted by the inter-AI constants program.
This exponential form is structurally consistent with both regimes
(τm → τ₀ at κ = 0, τm → ∞ as κ → 1).

field memory is demonstrated to be κ-independent —
i.e., high-κ and low-κ systems show identical memory duration.
